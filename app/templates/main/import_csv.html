Import Vials from CSV

CSV Format Instructions

How to use this feature:

  1. Go to Inventory Summary page.
  2. Click the "Export CSV" button to download the current inventory.
  3. Open the downloaded CSV file and make your desired changes to the data.
  4. Upload the modified CSV file here to update the database.

Supported operations:

  • Update existing records: Modify Status, Date Frozen, or Location
  • Add new records: Leave "Vial ID" empty for new vials

For new records, you must provide:

  • Batch Name: Name of the batch (will be created if doesn't exist)
  • Vial Tag: Unique identifier for the vial
  • Cell Line: Must be an existing cell line in the database
  • Location: Valid storage location (Tower/Drawer/Box R#C#)
  • Date Frozen: YYYY-MM-DD format (optional, defaults to today)
  • Status: Optional, defaults to "Available"

Additional fields you can update/add:

  • Passage Number: Cell passage information
  • Volume (ml): Volume in milliliters (numbers only)
  • Concentration: Cell concentration
  • Fluorescence Tag: Fluorescent protein tags
  • Resistance: Antibiotic resistance markers
  • Parental Cell Line: Original cell line information
  • Notes: Additional comments or information

Important: Do not modify the header row. For existing records, do not change the Vial ID or Vial Tag columns.
