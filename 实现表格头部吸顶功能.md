# 实现表格头部吸顶功能

## Core Features

- 表格滚动时表头固定在顶部，方便数据查阅

## Tech Stack

{
  "Web": {
    "arch": "html",
    "description": "采用纯CSS方案，在现有项目中为表格头（th）添加 `position: sticky` 样式，无需修改后端或引入JavaScript库。"
  }
}

## Design

在用户滚动页面时，表格头部将平滑地固定在视口顶部。设计上会为表头设置不透明背景和细微的底部阴影，以在视觉上将其与下方滚动的数据清晰分离，提升长数据表的可用性和阅读体验。
